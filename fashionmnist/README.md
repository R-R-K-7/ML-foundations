## Introduction 
* A model trained on FashionMNIST dataset using Pytorch to classify clothes based on their images.

## Dependencies
* `torch`
* `torchvision`
* `matplotlib`

## Methodology
* Dataset - FashionMNIST (28 × 28 grayscale images covering 10 clothing categories)
* Architecture - A custom CNN architecture utilizing standard convolutional layers paired with ReLU activation functions for non-linear feature extraction.

## Observation & Results
* Accuracy on test dataset : 82.13%
* Average loss on test dataset : 0.49
* A good accuracy on the test dataset shows that the model generalises well to unseen data.
