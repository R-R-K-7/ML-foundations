## Introduction
* This repository contains a number of mini machine learning projects. This demonstrates a dual focus on applied computer vision and machine learning fundamentals.
## Contents
### Fundamentals
* `./implementations` : Implementations of `KMeansClustering`, `DecisionTrees`, `PrincipalComponentAnalysis(PCA)`, `NeuralNetworks` entirely using numpy.<br>
### Computer Vision
* `./RockPaperScissors` : Real-time hand gesture detection and classification using YOLOv8n.<br>
* `./fashionmnist` : Custom CNN model for FashionMNIST dataset using PyTorch.<br>
* `./cifar100` : Custom CNN model for CIFAR100 dataset using PyTorch.<br>
