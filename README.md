## Introduction
* This repository contains a number of mini machine learning projects. This demonstrates a dual focus on applied computer vision and machine learning fundamentals.

## Contents

### Fundamentals
**`./implementations`**
* Implementations of `KMeansClustering`, `DecisionTrees`, `PrincipalComponentAnalysis(PCA)`, `NeuralNetworks` entirely using numpy.

### Computer Vision
**`./RockPaperScissors`**
* Real-time hand gesture detection and classification using YOLOv8n.
**`./fashionmnist`**
* Custom CNN model for FashionMNIST dataset using PyTorch.
**`./cifar100`**
* Custom CNN model for CIFAR100 dataset using PyTorch.
